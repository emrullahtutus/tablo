Import data

Here you can import your data into Katana using our templates. You can also update existing data in bulk using spreadsheets.

Add new products

You can import new products with initial stock amounts using our import template. Download and fill in the template and upload it back to Katana

Add new materials

You can import new materials with initial stock amounts using our import template. Download and fill in the template and upload it back to Katana.

Add new product recipes / BOMs

You can import recipes / BOMs to existing products in Katana using our import template. Download and fill in the template and upload it back to Katana to add new ingredients to products in Katana.

Add new product operations

You can import product operations to existing products in Katana using our import template. Download and fill in the template and upload it back to Katana to add new operations to products in Katana.

Add new batch / lot numbers

For adding batch numbers and their initial stock quantities for existing products and materials you get your own data as a template. Edit the data in the template and upload it back into Katana.

Add new sales orders

You can import sales orders from Google Sheets via Zapier. Once you have connected your Katana account and spreadsheet in Zapier, every new row added to a Google Sheets spreadsheet will be imported as a sales order in Katana.

Add new customers

You can import customers from Google Sheets via Zapier. Once you have connected your Katana account and spreadsheet in Zapier, every new row added to a Google Sheets spreadsheet will be imported as a customer in Katana.

Update existing products and reorder points

For updating existing products and reorder points you get your own data as a template. Edit the data in the template and then upload it back into Katana.

Update existing materials and reorder points

For updating existing materials and reorder points you get your own data as a template. Edit the data in the template and then upload it back into Katana.

Update stock levels and stock values

For updating stock levels and values for existing products and materials you get your own data as a template. Edit the data in the template and then upload it back into Katana. Perfect for stocktaking.

Cannot find an import you need? Configure your own using Katana API.
